# Simulation and Games

## Sebastian Andres Manotas Blanco
### EJERCICIO 1: Simular lanzamiento de dados
 Razonamiento: Genero números aleatorios entre 1 y 6 para simular dados. Puedo simular múltiples dados y múltiples lanzamientos.
### EJERCICIO 2: Simular Piedra-Papel-Tijera
Razonamiento: El jugador elige, la computadora elige aleatoriamente, y determinamos el ganador según las reglas clásicas.
### EJERCICIO 3: Juego de adivinar número
Razonamiento: La computadora piensa un número aleatorio entre 1 y 100. El jugador intenta adivinarlo y recibe pistas (mayor/menor).
### EJERCICIO 4: Generar tripletas pitagóricas
Razonamiento: Una tripleta pitagórica cumple: a² + b² = c². Genero y busco estas tripletas hasta un límite dado.
### EJERCICIO 5: Generador de números aleatorios (uniforme y normal)
Razonamiento: 
 - Distribución uniforme: todos los números tienen la misma probabilidad
 - Distribución normal: números cerca de la media son más probables
